## Get started

## 1. 创建文件夹

> mkdir FolderName

## 2. 进入文件夹

> cd FolderName

## 3. 下载模板

> git clone <https://github.com/mekefly/templates.git> -b typescript-rollup-vitest ./

> git branch -m main

thisPath 代表当前仓库的连接，你可以直接在上面地址栏复制

## 4. 安装依赖

> pnpm install

## 5. 删除本仓库地址

> git remote remove origin
